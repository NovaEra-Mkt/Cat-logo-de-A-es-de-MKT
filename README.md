# Cat-logo-de-A-es-de-MKT
Catálogo de Ações de Marketing é uma ferramenta colaborativa para registrar, organizar e acompanhar campanhas e iniciativas de marketing no varejo supermercadista. Permite cadastrar ações com budget, ROI, objetivos e anexos, filtrar por empresa, tipo, mês e status, além de exportar tudo para Excel.
